About the Project

The Employee Attrition Predictor leverages advanced machine learning algorithms to forecast which employees may leave. With this tool, organizations can analyze attrition trends, uncover root causes, and retain valuable talent more effectively.

Key Features

Accurate Predictions

Powered by XGBoost, delivering up to 75% prediction accuracy using real-world HR data.

Comprehensive Analysis

Analyze key factors like satisfaction, work environment, and compensation.

User-Friendly Interface

Built with HR professionals in mind – no technical background required.

Secure & Scalable

Secure authentication with clean routing using Django’s backend infrastructure.
